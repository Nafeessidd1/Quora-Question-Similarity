# **Quora Question Similarity Predictor**

Welcome to the **Quora Question Similarity Predictor**, a Flask web application designed to predict the similarity between pairs of questions. This application leverages Natural Language Processing (NLP) techniques and machine learning models to determine if two questions convey the same meaning.

## **Features**

- **User-Friendly Interface:** Simple and intuitive web interface for inputting question pairs.
- **Advanced Model:** Utilizes a pre-trained BERT model to predict question similarity.
- **Detailed Visualization:** Displays the similarity score and indicates whether the questions are similar.
