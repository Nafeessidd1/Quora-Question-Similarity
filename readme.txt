# **Quora Question Similarity Predictor**

This project is designed to predict the similarity between pairs of questions, specifically tailored for Quora questions. It includes a Flask web application that allows users to input pairs of questions and check their similarity. The project leverages Natural Language Processing (NLP) techniques and a pre-trained BERT model.

## **Tech Stack**

- **Programming Language**: Python
- **Libraries**:
  - Flask
  - TensorFlow
  - Transformers (Hugging Face)
  - Scikit-learn
  - Pandas
  - NumPy
- **Frontend**:
  - HTML
  - CSS
  - JavaScript
- **Tools**:
  - Jupyter Notebook
- **Dataset**:
  - Quora Question Pairs (from Kaggle)

## **Project Overview**

This project aims to predict the similarity between pairs of questions by analyzing their semantic meaning. The core of the project is a pre-trained BERT model fine-tuned on the Quora Question Pairs dataset. The project includes a Flask-based web application where users can input pairs of questions and receive real-time feedback on their similarity.
